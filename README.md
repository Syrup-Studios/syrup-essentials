### Features

* **Teleportation Suite(WIP):** Standard `/tpa`, `/tpahere`, `/warp`, and `/back` commands with configurable warmups and cooldowns.
* **Home Management:** Set multiple homes with `/sethome` and navigate easily with `/home`.
* **Essential Utilities(WIP):** Built-in support for `/spawn`, `/feed`, `/heal`, and `/fly`.
* **Optimized Performance(WIP):** Built from the ground up to be more resource-efficient than traditional utility mods.
* **Highly Configurable(WIP):** Extensive configuration abilities.
* A lot more planned!
